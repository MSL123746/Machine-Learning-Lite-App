# StreamLite Machine Learning App

A simple, interactive Streamlit app for training and evaluating machine learning models on tabular data.

## Features
- Upload CSV data
- Select model type: Regression, Binary Classification, Multi-class Classification, Clustering
- Choose features and target
- Configure training settings and hyperparameters
- Automatic data preprocessing
- Model training and evaluation
- Visualize results and metrics
- Modern, user-friendly UI

## Getting Started

### Prerequisites
- Python 3.8+
- See `requirements.txt` for dependencies

### Running Locally
```bash
pip install -r requirements.txt
streamlit run streamlit_ml_lite.py
```

### Deployment
- Deploy directly to [Streamlit Cloud](https://streamlit.io/cloud) or any platform that supports Streamlit apps.

## File Structure
- `streamlit_ml_lite.py` — Main application file
- `requirements.txt` — Python dependencies
- `uploads/` — (Optional) Folder for uploaded files
- `static/` — (Optional) Folder for static assets
